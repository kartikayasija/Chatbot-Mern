## Tech Stack

**Client:** React JS,SCSS, Typescript

**Server:** Node JS, Express JS, JWT, Socket.io

**Database:** Mongo DB
  

## Project initialization

Clone the project

```bash
  git clone https://github.com/kartikayasija/Chatbot-Mern
```
Install dependencies

```bash
  npm install
```

```bash

  npm install
```
